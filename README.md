# SnakeGame

--about--

  The classic snake game with a simple GUI built using pygame.
  
  
--modules/libraries used--
  
  -pygame
  
  -time
  
  -random
  
  
--system requirements--

-Windows 10/11

-Python 3.9.1

-IDE/VS Code

-1024x576p display

-2GB memory
